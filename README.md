# 🚀 Cari Kerja - Frontend Portal Lowongan Pekerjaan

A modern, high-performance frontend for a professional Job Portal, engineered with Next.js and Tailwind CSS. It features a unique **Master Profile** architecture that allows candidates to generate dynamically tailored, ATS-friendly resumes.

## ✨ Key Features
- **Master Profile vs Tailored Resumes**: Candidates can maintain a master database of their experiences, educations, and certifications, and cherry-pick them to create specific resumes.
- **ATS-Friendly CV Builder**: Generates professional PDF resumes directly from the browser using the tailored profile data.
- **Responsive Dashboard**: Beautiful, dark-mode compatible UI built with Tailwind CSS and modern web design principles.
- **Role-Based Workflows**: Seamlessly handles authentication and dashboards for both Job Seekers and Employers.

## 🛠️ Tech Stack
- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Data Fetching**: [React Query (TanStack)](https://tanstack.com/query) & Axios
- **UI Components**: Radix UI primitives & Custom glassmorphism UI
